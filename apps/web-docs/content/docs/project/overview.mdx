# Project Overview 项目总览

Project Overview 项目总览用于说明整套框架的全局目标、阶段目标与跟进方式。

这个项目不是单纯脚手架，而是一个本地优先的开发操作系统。

```text
Global Goal
  -> Initiatives
    -> Features
      -> Requirements
        -> Tasks
          -> Progress / Review / Release
```

## 项目边界

- 文档控制平面位于 `apps/web-docs/content/docs`
- 未来 CLI 与自动化代码位于 `apps/oc-pi-cli/src`
- 当前阶段先固定对象模型、分解规则与文档协议
