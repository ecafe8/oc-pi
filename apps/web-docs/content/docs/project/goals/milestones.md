# Milestones

## M1: Docs Protocol Stabilized

- 固定全局层、feature 层、执行层对象模型
- 固定 docs 目录结构与 front matter 规范
- 明确 `goal -> initiative -> feature -> task` 分解规则

## M2: Feature Planning MVP

- 支持创建 feature workspace
- 支持 feature version 文档骨架
- 支持 requirement 与 task graph 文档化

## M3: Progress Aggregation MVP

- 支持 feature progress 摘要
- 支持全局状态聚合
- 支持 blocker 与 next focus 输出

## M4: PI Operational Loops MVP

- 支持全局规划 loop
- 支持 feature 执行 loop
- 支持 review loop 的输出摘要

## M5: CLI-Driven Workflow

- 在 `apps/oc-pi-cli/src` 落地 CLI 入口
- 让 CLI 能读取这套文档协议并驱动循环
