# Docs-Native Control Plane

## Outcome

把文档空间从“展示层”提升为“人机共用控制平面”。

## Planned Features

- `create-feature-workspace`
- `manage-feature-metadata`
- `manage-prd-version`
- `render-feature-navigation`

## Notes

- narrative 内容与 structured control data 分层
- `front matter` 用于页面级元数据
- 结构化协议建议用 YAML 片段或后续独立数据文件表达
