# Task and Progress Control Plane

## Outcome

让系统不止能写 PRD，还能稳定地跟踪执行状态与阶段性产出。

## Planned Features

- `derive-requirements-from-prd`
- `generate-task-graph`
- `track-feature-progress`
- `aggregate-global-progress`

## Notes

- 任务必须支持依赖关系，而不是只有 checklist
- 进度记录应独立于任务定义，避免执行历史污染任务真源
