# Project Bootstrap

## Outcome

提供统一项目初始化入口，生成默认 monorepo 结构与基础应用组合。

## Planned Features

- `init-monorepo-project`
- `init-default-apps`
- `init-docs-workspace`

## Notes

- 输出结构默认包含 `apps/server-api`、`apps/web-app`、`apps/web-docs`
- 未来 CLI 的初始化命令将落在 `apps/oc-pi-cli/src`
