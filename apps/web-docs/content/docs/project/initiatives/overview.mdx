# Initiatives Overview

`initiative` 不是具体功能，而是一类需要持续建设的能力主题。

## Current Initiative Set

- [Project Bootstrap](./project-bootstrap)
- [Docs-Native Control Plane](./docs-native-control-plane)
- [Task and Progress Control Plane](./task-and-progress-control-plane)
- [PI Operational Loops](./pi-operational-loops)

## Initiative Rule

一个 `initiative` 应满足：

- 本身还不是最终交付物
- 下游通常可拆成 `2-5` 个 feature
- 关注一类能力建设，而不是一组离散任务
