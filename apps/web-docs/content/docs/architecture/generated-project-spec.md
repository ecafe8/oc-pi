# Generated Project Spec 生成项目规格草案

Generated Project Spec 生成项目规格草案用于记录未来产品输出方向，但当前不把它当作已完成规范。

## Purpose 目的

- 为后续模板系统提供目标轮廓
- 为后续用户项目初始化能力提供讨论基线
- 避免当前产品规划和未来输出结构混淆

## Draft Scope 草案范围

- 未来用户项目的 monorepo 结构
- 未来用户项目的 `web-docs` 控制平面
- 未来用户项目的 feature 与 task 组织方式

## Current Rule 当前规则

- 本文档只记录草案，不作为当前实现输入真源
