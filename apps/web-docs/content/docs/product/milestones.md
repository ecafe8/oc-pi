# Milestones 里程碑

Milestones 里程碑用于定义产品研发推进中的阶段性完成标准。

## M1: Product Planning Stable 产品规划稳定

- 愿景、能力地图、架构草案与任务模型被写清楚
- 当前文档空间能稳定服务 `apps/oc-pi-cli` 的规划讨论

## M2: Runtime Skeleton Ready 运行时骨架可用

- `apps/oc-pi-cli/src` 内有清晰的模块边界
- 核心 loop 与 orchestration 编排骨架被定义

## M3: Template Engine Draft 模板引擎草案完成

- 明确模板规范
- 明确模板变量与输出结构
- 明确哪些模板服务产品自身，哪些模板服务未来用户项目

## M4: Docs and Task Loop MVP 文档与任务循环 MVP

- 可读取规划文档
- 可输出任务建议
- 可回写进度摘要

## M5: Generated Project Spec Ready 生成项目规格可用

- 未来用户项目的 `web-docs` 与 monorepo 协议形成可实现草案
