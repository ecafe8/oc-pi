# Protocols Overview 协议总览

Protocols 协议层用于定义这套 docs-native 文档原生控制平面的核心规则。

协议层的目标是回答三个问题：

- 系统里有哪些一等对象
- 它们如何分解与关联
- 文档空间如何表达这些对象
