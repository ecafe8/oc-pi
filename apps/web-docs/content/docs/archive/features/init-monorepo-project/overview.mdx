# Init Monorepo Project 初始化 Monorepo 项目

Init Monorepo Project 初始化 Monorepo 项目是一个 feature 功能单元，用于定义默认项目骨架如何被创建。

## Meta 元数据

Meta 元数据用于描述该 feature 功能单元的标识、所属 initiative 计划主题、状态与实现落点。

```yaml
feature:
  id: init-monorepo-project # 唯一标识
  initiative: project-bootstrap # 所属计划主题
  status: definition # 当前状态
  owner: pi # 责任主体
  implementationRoot: apps/oc-pi-cli/src # 实现目录
```
