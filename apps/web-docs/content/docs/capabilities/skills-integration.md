# Skills Integration Skills 集成

Skills Integration Skills 集成用于定义系统如何发现、选择并调用 Skills 技能能力。

## Scope 范围

- Skills 技能索引与选择
- Skills 技能调用编排
- Skills 技能结果与 loop、文档、代码输出联动

## Planned Features 规划中的功能单元

- Skills 发现
- Skills 选择策略
- Skills 调用结果整合
