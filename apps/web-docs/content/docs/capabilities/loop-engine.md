# Loop Engine 循环引擎

Loop Engine 循环引擎用于定义系统如何周期性读取上下文、识别缺口并输出下一步动作。

## Scope 范围

- 全局 planning loop 规划循环
- feature execution loop 功能执行循环
- review loop 复核循环
- release loop 发布循环

## Planned Features 规划中的功能单元

- loop 输入输出契约
- loop 调度器
- loop 结果回写
