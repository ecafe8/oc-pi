# Conversation Orchestration 对话编排

Conversation Orchestration 对话编排用于定义系统如何管理一轮或多轮 AI 对话，以及如何把对话转成后续动作。

## Scope 范围

- 管理对话上下文
- 管理用户意图到执行动作的衔接
- 将讨论结果转化为任务、文档或后续 loop 输入

## Planned Features 规划中的功能单元

- 对话会话管理
- 意图识别与动作分发
- 对话结果摘要与沉淀
