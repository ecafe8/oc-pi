# Capabilities Overview 能力总览

Capabilities Overview 能力总览用于定义 `apps/oc-pi-cli` 作为产品执行引擎需要承担的一级能力模块。

## Capability Map 能力地图

- [Conversation Orchestration 对话编排](./conversation-orchestration)
- [Loop Engine 循环引擎](./loop-engine)
- [Template Engine 模板引擎](./template-engine)
- [Docs Generation 文档生成](./docs-generation)
- [Code Generation 代码生成](./code-generation)
- [MCP Integration MCP 集成](./mcp-integration)
- [Skills Integration Skills 集成](./skills-integration)

## Rule 规则

- 能力模块用于描述一级产品能力，而不是单条任务
- 每个能力模块后续都可以继续拆成独立 feature 功能单元

## Capability to MVP Mapping 能力到 MVP 映射

Capability to MVP Mapping 能力到 MVP 映射用于说明一级能力模块如何服务当前第一批 MVP 功能。

| Capability 能力模块 | Supports MVP Features 支持的 MVP 功能 |
|---|---|
| Conversation Orchestration 对话编排 | `goal-to-docs`、`interactive-workbench` |
| Loop Engine 循环引擎 | `review-loop` |
| Template Engine 模板引擎 | `project-bootstrap`、`goal-to-docs` |
| Docs Generation 文档生成 | `goal-to-docs` |
| Code Generation 代码生成 | 后续 `code-writer 代码编写者` 路径 |
| MCP Integration MCP 集成 | 后续扩展，不在当前 MVP 主闭环 |
| Skills Integration Skills 集成 | 作为基础设施支撑 `goal-to-docs` 与后续扩展 |
| Context Management 上下文管理 | `goal-to-docs`、`review-loop`、`interactive-workbench` |

## Capability Dependency Sketch 能力依赖草图

Capability Dependency Sketch 能力依赖草图用于说明当前能力模块之间的主干依赖关系。

```text
Conversation Orchestration 对话编排
  -> Context Management 上下文管理
  -> Docs Generation 文档生成

Template Engine 模板引擎
  -> Project Bootstrap 项目初始化
  -> Docs Generation 文档生成

Loop Engine 循环引擎
  -> Context Management 上下文管理
  -> Docs Generation 文档生成

Interactive Workbench 交互工作台
  -> Conversation Orchestration 对话编排
  -> Context Management 上下文管理
```
